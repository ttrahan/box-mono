# Box
A cron application which sends a pulse to a database regularly.

#### Environment Variables Required:
 - DB_USERNAME
 - DB_PASSWORD
 - DB_HOST
 - DB_PORT
 - DB_NAME
 - ENVIRONMENT

This project works together with [Data Visualization][1] project.
